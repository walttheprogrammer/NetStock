# NetStock
Aplicación de gestión de inventarios para bodegas locales. Implementa administración de ventas, manejo de precios, control de stock y un listado de clientes fiados (cuentas por cobrar), además de tener una interfaz (UX) personalizable a las preferencias del usuario. Se desarrolló para una bodega local ubicada en Los Olivos, de nombre "El Olivar".
